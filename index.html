Métodos Básicos de Arrays em JavaScript

Apresentação dos principais métodos para manipulação de arrays.

Array.length

Retorna o número de elementos em um array.

const frutas = ["maçã", "banana"];
console.log(frutas.length); // 2

Array.toString()

Converte o array em uma string separada por vírgulas.

const frutas = ["maçã", "banana"];
console.log(frutas.toString()); // "maçã,banana"

Array.at()

Retorna o elemento na posição especificada. Aceita índices negativos.

const frutas = ["maçã", "banana", "laranja"];
console.log(frutas.at(-1)); // "laranja"

Array.join()

Junta todos os elementos do array usando um separador.

const frutas = ["maçã", "banana"];
console.log(frutas.join(" - ")); // "maçã - banana"

Array.pop()

Remove o último elemento do array e o retorna.

const frutas = ["maçã", "banana"];
console.log(frutas.pop()); // "banana"

Array.push()

Adiciona um ou mais elementos ao final do array.

const frutas = ["maçã"];
frutas.push("banana");
console.log(frutas); // ["maçã", "banana"]
